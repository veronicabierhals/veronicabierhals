Portfólio

API Ponto de Venda

Projetada para simplificar operações de PDV. Desde o cadastro de usuários até o gerenciamento de perfil, nossa API oferece uma série de recursos essenciais.
Tecnologias utilizadas: node.js, Express (v4.18.2), PostgreSQL (pg v8.11.3), BCrypt (v5.1.1), JSON Web Token (JWT v9.0.2), Dotenv (v16.3.1), Cors (v2.8.5), Joi (v17.11.0), AWS SDK (v2.1516.0), Multer (v1.4.5-lts.1), Nodemailer (v6.9.7), Knex (v3.0.1), ElephantSQL para armazenamento de dados.

Repositório no GitHub

API de controle financeiro

A API oferece funcionalidades incluindo cadastro e autenticação de usuários, gerenciamento de categorias, registro de transações financeiras (entradas e saídas) e a obtenção de extrato financeiro.
Tecnologias Utilizadas: Node.js, Express.js, PostgreSQL (banco de dados), Bcrypt (para criptografia de senhas), JSON Web Token (JWT - para autenticação via token), dotenv (Para configuração de variáveis de ambiente), Insomnia (para testes).

Repositório no GitHub

Mini Rede Social

Projeto de API simples inspirado no Instagram, onde os usuários podem: realizar postagens, curtir postagens, comentar e muito mais.
Tecnologias Utilizadas: Node.js, Express e PostgreSQL.
